How to

Abstract

Purpose of this site is to showcase - calling of multiple APIs asynchronously and progressively rendering in html using Marko.js templating engine

Outcome

Performance Improvements - User sees NO wait / NO blank screen even on calling multiple APIs

Knome post:
  • Link

    • Source Code:
  • https://github.com/prabusah/AsyncAPI-Marko
  • Goto AsyncAPI-Marko/home/index.js - Replace with YOUR ConsumerKey / ConsumerSecret from APPhonics
  • Goto AsyncAPI-Marko/api/index.js - Replace with YOUR APIKey from data.gov.in

    • Install below components:

  • bower
  • gulp
  • node.js

  • bower install typeahead.js

  • bower install jquery
  • npm install
  • gulp
  • node server.js
